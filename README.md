## About Me

I am a Computer Science student at the University of Colorado Boulder focused on software engineering, AI/ML systems, distributed data platforms, and developer tooling. I like building practical systems that make complex data easier to understand, whether that means real-time analytics pipelines, internal engineering dashboards, mobile support communities, or ML models deployed across distributed environments.

Previously, I worked as a Software Engineering Intern at Empower, where I built an interactive API dependency network graph used by 200+ engineers to visualize service relationships, ownership, and health metadata. I have also contributed to AI and cybersecurity research on federated learning-enabled anomaly detection for drone swarms, with work accepted at ICDCN 2025.

## Featured Public Projects

### [Real-Time Event Streaming & Analytics Platform](https://github.com/rohanadepu/Real-Time-Event-Streaming-Analytics-Platform)

Production-style analytics platform built with Kafka, Flink/Spark, Redis, and FastAPI for high-throughput event processing, low-latency dashboards, anomaly detection, and observability.

- Processed 5,000+ events/sec with p95 latency under 150ms
- Added z-score and MAD anomaly detection for operational metrics
- Integrated Prometheus and Grafana monitoring
- Containerized services with Docker and designed for Kubernetes deployment

### [Personal Finance Data Aggregator & Insights Platform](https://github.com/rohanadepu/-Personal-Finance-Data-Aggregator-Insights-Platform)

End-to-end financial analytics platform using Airflow, Spark, PostgreSQL, Streamlit, and ML forecasting models.

- Processed 10M+ records through automated ETL pipelines
- Built anomaly detection and forecasting models using scikit-learn, Prophet, ARIMA, and LSTM
- Delivered interactive dashboards for financial insights and KPI tracking
